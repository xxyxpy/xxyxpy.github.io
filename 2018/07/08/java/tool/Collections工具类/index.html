生成空的List、Set、Map

批量添加addAll

public static boolean addAll(Collection<? super T> c, T… elements)

排序sort

public static > void sort(List list) 需要泛型T类本身支持Comparable接口
public static void sort(List list, Comparator<? super T> c)

使用方法见下面的二分查找

二分查找binarySearch

public static int binarySearch(List<? extends Comparable<? super T>> list, T key)
public static int binarySearch(List<? extends T> list, T key, Comparator<? super T> c)

在执行查找之前需要先对list进行升序排序操作

混排shuffle

public static void shuffle(List<?> list）
public static void shuffle(List<?> list, Random rnd)

混排算法用于打乱List中可能有的任何排列的踪迹，适合用于随机性的需求

反转reverse

public static void reverse(List<?> list)直接反转集合的元素

public static Comparator reverseOrder()；返回可以使集合反转的比较器Comparator

public static Comparator reverseOrder(Comparator cmp)；如果cmp不为null，返回cmp的反转的比较器，即集合的反转的反转，最后就是没反转。如果cmp为null，效果等同于第二个方法.

线程安全集合synchronized系列

public static Collection synchronizedCollection(Collection c)

public static SortedSet synchronizedSortedSet(SortedSet s)

public static List synchronizedList(List list)

public static Map synchronizedMap(Map m)

public static SortedMap synchronizedSortedMap(SortedMap m)

public static SortedSet synchronizedSortedSet(SortedSet s)

传入普通的集合，通过包装的方式返回结束安全的集合，本质上还是在相关的操作上添加synchronized关键字。

不可变集合unmodifiable系列

public static Collection unmodifiableCollection(Collection<? extends T> c)

public static Set unmodifiableSet(Set<? extends T> s)

public static SortedSet unmodifiableSortedSet(SortedSet s)

public static List unmodifiableList(List<? extends T> list)

public static Map unmodifiableMap(Map<? extends K, ? extends V> m)

public static SortedMap unmodifiableSortedMap(SortedMap m)

实现不可变集合的本质是将所有改变容器状态的操作（put add remove等等）都包装成未实现，调用时直接返回异常

元素交换swap

交换List中两个指定位置的元素

public static void swap(List<?> list, int i, int j)

填充容器fill

public static void fill(List<? super T> list, T obj)

使用指定数值填充容器

复制集合copy

public static void copy(List<? super T> dest, List<? extends T> src)

复制完成后索引值一一对应，目标集合的大小必须大于源集合

替换replaceAll

public static boolean replaceAll(List list, T oldVal, T newVal)

public static T min(Collection<? extends T> coll, Comparator<? super T> comp)

public static > T max(Collection<? extends T> coll)

public static T max(Collection<? extends T> coll, Comparator<? super T> comp)
